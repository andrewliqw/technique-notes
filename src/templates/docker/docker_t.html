What is docker?

One thing to remember: the docker container itself is Linux program. At this moment, there is no Windows program can be wrapped in the docker container.

Docker command

docker version: display version

docker images: list images.

docker pull: downlaod image.

docker ps: list containers.

docker run: run command in container

docker stop: stop running a container

docker start: strat running a container

Push your image to hub.docker.com

Two steps:

$ docker login

$ docker push andrewliqw/image-name-and-tag

Postgresql

installation

$ docker pull postgres

create a container of postgre server and run at the frist time

$ docker run --name postgres-andrew -d postgres

create a container to connect to postgre server

$ docker run -it --rm --link postgre-andrew:postgres postgres psql -h postgres -U postgres
