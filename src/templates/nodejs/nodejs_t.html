Installation

Mac OS X

For Mac, download the package node-v7.2.1.pkg, double click to install it

After installation, update npm because node package install lower version npm.
$ sudo npm install npm@latest -g

Source code

$ ./configure --prefix=/usr/local/nodev7.7.1

$ make -j4

$ sudo make install

$ /usr/local/nodev7.7.1/bin/npm config set prefix ~/npm-global

$ /usr/local/nodev7.7.1/bin/npm install npm@latest -g

$ export PATH=~/npm-global/bin:$PATH

Interactive shell

Enter "node" command directly enter into interactive shell. It is colorful by default.
$ node
