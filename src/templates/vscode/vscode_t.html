Where is the user settings location

For Mac, it is in the file ~/Library/Application\ Support/Code/User/settings.json

Vertical rulers

editor.rulers: [80, 120]

comment out python code

Select the code to be commented, press ⌘ + k, then ⌘ + c.

⌘ + u to remove the comment.

column selection

Column selection means select a rectangle range. ⇧ + ⌘ + ⌥ + arrow

split editor window

Using ⌘ + \ to split the window horizontally. Maximum editor window is 3. Using ⌘ + 1 or ⌘ + 2 or ⌘ + 3 to switch to any window.
