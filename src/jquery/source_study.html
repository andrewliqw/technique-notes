src/var directory

The src/var directory includes some small javascript modules which can be exported to other javascript files using AMD define function.

src/var/class2type.js

This module exports class2type = {} to the including context.

define(["var/class2type"], function(class2type){
    // your code can reference class2type directly
    // equal to: var class2type = {}
});

src/var/toString.js

This module exports toString = ({}).toString to the including context.

        define(["var/toString"], function(toString){
            // your code can reference toString directly
            // equal to: var toString = ({}).toString
        });

src/core.js

jQuery.extend function

There are two forms to call this function

  • jQuery.extend(object1, object2, ..., objectn)
  • jQuery.extend(true, object1, object2, ..., objectn)

If there is only one argument, then the calling object is extended.

        jQuery.extend({apple: 0, banana: { weight: 52, price: 100 }, cherry: 97}, {banana: { price: 200 }, durian: 100});
        jQuery.extend(true, {apple: 0, banana: { weight: 52, price: 100 }, cherry: 97}, {banana: { price: 200 }, durian: 100});
